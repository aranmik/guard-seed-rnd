# Guard Seed × Quarterview R&D — QV01 Tuning Notes

나라가 폰에서 손맛 값을 찾고 기록하는 노트. 최종 확정값은 [GUARD_SEED_QV01_HANDOFF.md](GUARD_SEED_QV01_HANDOFF.md)로 넘어가 다음 패스의 기본값이 된다.

---

## 1. 사용 방법

1. 폰에서 Pages(`https://aranmik.github.io/guard-seed-rnd/`) 접속
2. 전장 또는 보스 듀얼 진입
3. FEEL DEV(⚙) 열기
4. 값 조정
5. 마음에 들면 ★ Snapshot 저장
6. Copy Current로 값 복사
7. 이 문서에 붙여넣거나 유키/렌에게 전달

## 2. 튜닝 원칙

- 최종 기준은 폰 실기
- 노트북은 참고만
- 너무 쉬운 값보다 "읽고 성공했을 때 기쁜 값" 우선
- Guard는 허용하되 Chance는 Perfect 전용
- 입력이 밀리거나 답답하면 Pose Hold / Input Lock / Hitstop 먼저 확인
- 링이 불쾌하면 Speed보다 예고/힌트/시인성을 먼저 확인
- 멈추는 링/훼이크 링은 반드시 대응 가능한 힌트가 있어야 한다
